# ガイダンス

**日越大学におけるVJU2020・VJU2021正規課程学部学生を対象とした、外国語証明書・認定書の卒業基準確認及び外国語科目運営における使用について**

*2014年7月21日付首相決定第1186/QĐ-TTg号「ベトナム国家大学ハノイ校傘下の日越大学設立について」に基づき、*

*2017年3月31日付首相決定第391/QĐ-TTg号「ベトナム国家大学ハノイ校日越大学の組織及び運営に関する規程」に基づき、*

*2014年12月25日付ベトナム国家大学ハノイ校総長決定第5115/QĐ-ĐHQGHN号「ベトナム国家大学ハノイ校における学部教育に関する規程の公布について」に基づき、*

*2021年12月14日付ベトナム国家大学ハノイ校公文書第4116/ĐHQGHN-ĐT号「学部教育における外国語能力基準に関するガイダンス」に基づき、*

*2020年10月9日付日越大学学長ガイダンス第784/ĐHVN-ĐT号「学部教育課程における外国語科目の教育組織について」に基づき、*

*2024年3月14日付ベトナム国家大学ハノイ校総長公文書第861/ĐHQGHN-ĐT号「QH.2021以前の学生に対する外国語科目の単位認定について」に基づき、*

日越大学は、VJU2020・VJU2021正規課程学部学生を対象とした、卒業基準の確認及び外国語科目の運営における外国語証明書・認定書の使用に関するガイダンスを以下のとおり定める。

## 1. 適用対象

VJU2020及びVJU2021の正規課程学部学生。

## 2. 卒業基準の確認

卒業前に、学生は教育課程（CTĐT）の卒業基準に適合するベトナム外国語能力枠組みに従った有効な外国語能力の証明書又は認定書を提出しなければならない。具体的には以下のとおりである。

| 番号 | 教育課程 | ベトナム外国語能力枠組み | |
| :-- | :--- | :--- | :--- |
| | | **英語** | **日本語** |
| 1 | 日本研究 | レベル3 | レベル4 |
| 2 | コンピュータ科学・工学 | レベル4 | – |

- ベトナム国家大学ハノイ校で認められる証明書・認定書の一覧は、本ガイダンスの付録1を参照すること。
*注：有効な証明書・認定書は、受験日から2年間有効とする。*

## 3. 教育課程の枠組み内における外国語科目の学習成果の認定

### 3.1. コンピュータ科学・工学高品質教育課程

有効な英語の証明書・認定書を提出した学生は、一般知識ブロック（M1）の英語B1及び英語B2科目の学習成果の認定を以下のとおり受けることができる。

| 番号 | ベトナム外国語能力枠組み | 認定結果 | 認定される科目 | 科目コード |
| :-- | :--- | :--- | :--- | :--- |
| 1 | レベル3 | 合格 | 英語B1 | FLF1107 |
| 2 | レベル4 | 合格 | 英語B1、英語B2 | FLF1107、FLF1108 |

### 3.2. 日本研究高品質教育課程

**a) 英語**
- 認定対象は、専門知識ブロックに属する英語B1科目（FLF1107）に限り、VNU-Tests（ベトナム国家大学ハノイ校における職業コミュニケーション志向外国語能力評価試験）の認定書のみを適用する。その他の外国語証明書は対象外とする。
- VNU-Tests認定書に基づく学習成果の認定は以下のとおりである。

| 番号 | 試験形式 | 認定スコア |
| :-- | :--- | :--- |
| 1 | 英語レベル3 | 9.0 |
| 2 | 英語レベル4 | 10.0 |

**b) 日本語**
- 一般知識ブロック（M1）における日本語B1及び日本語B2科目については、2020年10月9日付ガイダンス第784/ĐHVN-ĐT号「学部教育課程における外国語科目の教育組織について」第3条第3.1項の規定に従うものとする。
- 認定対象となる証明書・認定書は、本ガイダンスの付録1を参照すること。

## 4. 留意すべき情報

- 外国語の認定書・証明書は、提出時点で有効期限内のものでなければならない。学生は、認定書・証明書の公証済みコピー1部を教務・学生課（ミーディンキャンパス501号室）に提出すること。JLPT受験者は、付録2の様式に従い証明書審査委任状を併せて提出する必要がある。
- 既に学長決定により卒業基準の達成が認定されている学生は、証明書・認定書の再提出を要しない。
- オンライン形式で受験した外国語の認定書・証明書は、日越大学では認められない。
- 日越大学は、卒業判定前に証明書・認定書の審査が間に合うよう、早期の提出を奨励する。
- 証明書・認定書の提出に関する質問は、教務・学生課（メール: phongdaotao@st.vju.ac.vn）まで直接問い合わせること。
